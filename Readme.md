# AI Data Analyst Assistant

An automated data analysis tool that performs exploratory data analysis and generates insights from any dataset.

## Features

- Dataset preview
- KPI dashboard summary
- Missing value analysis
- Column type detection
- Automatic chart generation
- Correlation heatmap
- Strong correlation insights
- Categorical value analysis
- Outlier detection
- Smart dataset cleaning
- Download cleaned dataset
- Download dataset analysis report

## Tech Stack

Python  
Pandas  
Streamlit  
Matplotlib  
Seaborn  

## How to Run

1. Clone the repository

git clone https://github.com/GeethaThimmaraju/AI-Data-Analyst-Assistant

2. Install dependencies

pip install -r requirements.txt

3. Run the app

streamlit run app.py
